# Folder Structure Guide

Create these folders in your repository:

```
/assets
  /img
    place your profile photo and other images here
  /css
    (optional) for any custom CSS
  /js
    (optional) for any custom JavaScript
  /files
    place your CV PDF and other downloadable files here
```

## Next Steps

1. Upload a profile photo to `/assets/img/` and name it `logo.png`
2. Update the placeholder text in all .md files with your actual information
3. Add any additional pages you might need
4. If you have a CV PDF, upload it to `/assets/files/`
